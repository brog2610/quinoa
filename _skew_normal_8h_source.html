00007   \brief     System of skew-normal SDEs
00008   \details   This file implements the time integration of a system of stochastic
00009     differential equations (SDEs), whose invariant is the joint [skew-normal
00010     distribution](http://en.wikipedia.org/wiki/Skew_normal_distribution).
00011 
00012     In a nutshell, the equation integrated governs a set of scalars,
00013     \f$x_\alpha\f$, \f$\alpha\!=\!1,\dots,N\f$, as
00014     \f[
00015        \mathrm{d}x_\alpha(t) = -\frac{1}{T_\alpha}\left[x_\alpha -
00016        \lambda_\alpha\sigma^2_\alpha\sqrt{\frac{2}{\pi}} \cdot
00017        \frac{\exp{\left(-\lambda_\alpha^2x^2_\alpha/2\right)}}{1+\mathrm{erf}
00018        \left( \lambda_\alpha x_\alpha/\sqrt{2}\right)} \right] \mathrm{d}t +
00019        \sqrt{\frac{2\sigma^2_\alpha}{T_\alpha}}\mathrm{d}W_\alpha(t).
00020     \f]
00021     The invariant distribution is the joint skew-normal distribution
00022     \f[
00023        p(x_\alpha) = \frac{1}{\sigma_\alpha\sqrt{2\pi}} \exp\left(
00024        -\frac{x^2_\alpha}{2\sigma^2_\alpha} \right) \left[1 + \mathrm{erf}\left(
00025        \frac{\lambda_\alpha x_\alpha}{\sqrt{2}}\right) \right].
00026     \f]
00027     Here \f$\mathrm{erf}(y) = 2/\sqrt{\pi} \int_0^y \exp(-u^2) \mathrm{d}u\f$,
00028     \f$T_\alpha\f$ are time scales, \f$\sigma_\alpha\f$ are variance parameters,
00029     \f$\mathrm{d}W_\alpha(t)\f$ is an isotropic [Wiener
00030     process](http://en.wikipedia.org/wiki/Wiener_process) with independent
00031     increments, and \f$\lambda_\alpha\f$ are the parameters that control the
00032     asymmetry and skewness for variable \f$x_\alpha\f$: \f$\lambda_\alpha<0\f$
00033     and \f$\lambda_\alpha>0\f$ give negative and positive skewness,
00034     respectively, while \f$\lambda_\alpha=0\f$ reduces the system to a set of
00035     independent Ornstein-Uhlenbeck processes. See also
00036     DiffEq/DiagOrnsteinUhlenbeck.h. For more details on the skew-normal
00037     distribution, see http://www.jstor.org/stable/2337278.
00038 */

00120           tk::real& x = particles( p, i, m_offset );
00121           tk::real d = 2.0 * m_sigmasq[i] / m_T[i] * dt;

00127                    ) / m_T[i] * dt
