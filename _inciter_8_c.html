Inciter main()

Inciter does have a main() function so that Zoltan, an MPI library, can partition the mesh. Then we initialize the Charm++ runtime system and do a calculation. This is necessary, since MPI_Init() is a bit adamant about capturing resources it wants and hence it has to be called before Charm is initialized.
