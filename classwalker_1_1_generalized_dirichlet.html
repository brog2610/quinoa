Detailed Description

template<class Init, class Coefficients>
class walker::GeneralizedDirichlet< Init, Coefficients >

Lochner's generalized Dirichlet SDE used polymorphically with DiffEq.

The template arguments specify policies and are used to configure the behavior of the class. The policies are:

  • Init - initialization policy, see DiffEq/InitPolicy.h
  • Coefficients - coefficients policy, see DiffEq/GeneralizedDirichletCoeffPolicy.h
