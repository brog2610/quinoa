1 ## What is Quinoa?
2 
3 <img src="http://quinoacomputing.github.io/quinoa/quinoa.svg" align="right" width="25%" background=transparent>
4 Quinoa is a set of computational tools that enables research and numerical analysis in fluid dynamics. At this time it is a test-bed to experiment with various algorithms using fully asynchronous runtime systems.
5 
6 ## Organization
7 
8 Currently, Quinoa consists of the following tools:
9  - [<B>inciter</B>](https://quinoacomputing.github.io/quinoa/inciter_doc.html) - Solve a PDE on an unstructured mesh using overdecomposition
10  - [<B>walker</B>](https://quinoacomputing.github.io/quinoa/walker_doc.html) - Random walker for stochastic differential equations
11  - [<B>rngtest</B>](https://quinoacomputing.github.io/quinoa/rngtest_doc.html) - Random number generator test suite
12  - [<B>unittest</B>](https://quinoacomputing.github.io/quinoa/unittest_doc.html) - Unit test suite
13  - [<B>meshconv</B>](https://quinoacomputing.github.io/quinoa/meshconv_doc.html) - Mesh file converter

35 ## Build
36 
37 ### 1. Install prerequisites
38 
39 - Debian/Ubuntu linux: (line 1: required, line 2: recommended)
40 
41  ```
42  sudo apt-get install git cmake gfortran gcc g++ openmpi-bin libopenmpi-dev
43  sudo apt-get install gmsh libpugixml-dev libpstreams-dev libboost-all-dev liblapack-dev liblapacke-dev libhdf5-dev libhdf5-openmpi-dev libhypre-dev
44  ```
45 
46 - Mac OS X: (line 1: required, line 2: recommended)
47 
48  ```
49  sudo port install git cmake openmpi-clang38
50  sudo port gmsh pugixml boost hdf5 +hl +openmpi hypre +openmpi
51  ```
52 
53 ### 2. Clone, build third-party libraries, build & test
54 
55  ```
56  git clone https://github.com/quinoacomputing/quinoa.git; cd quinoa
57  mkdir tpl/build; cd tpl/build; cmake ..; make; cd -
58  mkdir build; cd build; cmake ../src; make; ../script/run_tests.sh
59  ```
60 
61  - All executables will be in <tt>quinoa/build/Main</tt>
62  - Browse the [documentation](http://quinoacomputing.github.io/quinoa/index.html)
