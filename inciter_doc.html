Inciter: Solve a PDE on unstructured meshes

Inciter is used to experiment with asynchronous algorithms to numerically solve a system of partial differential equations discretized with unstructured meshes using overdecomposition. Similar to the rest of Quinoa, inciter also uses the Charm++ runtime system.

Help on command-line parameters

As usual, use the -h command-line parameter to get on-screen help from an executable. Example output, exluding Charm++ command-line parameters:

1 $ inciter -h
2 inciter Command-line Parameters:
3  -c, --control string Specify the control file name [REQUIRED]
4  -h, --help Display one-liner help on all command-line arguments
5  -f, --helpctr Display one-liner help on all control file keywords
6  -H, --helpkw string Display verbose help on a single keyword
7  -i, --input string Specify the input file
8  -o, --output string Specify the output file
9  -v, --verbose Select verbose screen output
10 -u, --virtualization real Set degree of virtualization
