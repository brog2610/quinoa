Detailed Description

Random number generator.

This class uses runtime polymorphism without client-side inheritance: inheritance is confined to the internals of the this class, inivisble to client-code. The class exclusively deals with ownership enabling client-side value semantics. Credit goes to Sean Parent at Adobe: https://github.com/sean-parent/sean-parent.github.com/wiki/ Papers-and-Presentations. For example client code that models a RNG, see see tk::MKLRNG or tk::RNGSSE.

Author:
J. Bakosi
