00004   \author    J. Bakosi

00007   \brief     Dirichlet SDE
00008   \details   This file implements the time integration of a system of stochastic
00009     differential equations (SDEs), whose invariant is the [Dirichlet
00010     distribution](http://en.wikipedia.org/wiki/Dirichlet_distribution).
00011 
00012     In a nutshell, the equation integrated governs a set of scalars,
00013     \f$0\!\le\!Y_\alpha\f$, \f$\alpha\!=\!1,\dots,N-1\f$,
00014     \f$\sum_{\alpha=1}^{N-1}Y_\alpha\!\le\!1\f$, as
00015     \f[
00016        \mathrm{d}Y_\alpha(t) = \frac{b_\alpha}{2}\big[S_\alpha Y_N -
00017        (1-S_\alpha)Y_\alpha\big]\mathrm{d}t + \sqrt{\kappa_\alpha Y_\alpha
00018        Y_N}\mathrm{d}W_\alpha(t), \qquad \alpha=1,\dots,N-1
00019     \f]
00020     with parameter vectors \f$b_\alpha > 0\f$, \f$\kappa_\alpha > 0\f$, and \f$0
00021     < S_\alpha < 1\f$, and \f$Y_N = 1-\sum_{\alpha=1}^{N-1}Y_\alpha\f$. Here
00022     \f$\mathrm{d}W_\alpha(t)\f$ is an isotropic vector-valued [Wiener
00023     process](http://en.wikipedia.org/wiki/Wiener_process) with independent
00024     increments. The invariant distribution is the Dirichlet distribution,
00025     provided the parameters of the drift and diffusion terms satisfy
00026     \f[
00027       (1-S_1) b_1 / \kappa_1 = \dots = (1-S_{N-1}) b_{N-1} / \kappa_{N-1}.
00028     \f]
00029     To keep the invariant distribution Dirichlet, the above constraint on the
00030     coefficients must be satisfied. For more details on the Dirichlet SDE, see
00031     http://dx.doi.org/10.1155/2013/842981.
00032 */
