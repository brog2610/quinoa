00002 /*!
00003   \file      src/DiffEq/MixNumberFractionBeta.h
00004   \author    J. Bakosi

00007   \brief     System of mix number-fraction beta SDEs
00008   \details   This file implements the time integration of a system of stochastic
00009     differential equations (SDEs) with linear drift and quadratic diagonal
00010     diffusion, whose invariant is the joint [beta
00011     distribution](http://en.wikipedia.org/wiki/Beta_distribution). There are two
00012     differences compared to the plain beta SDE (see DiffEq/Beta.h):
00013 
00014     - First, the parameters, b, and kappa are specified via functions that
00015     constrain the beta SDE to be consistent with the turbulent mixing process.
00016     In particular, the SDE is made consistent with the no-mix and fully mixed
00017     limits. See, e.g., MixNumberFractionBetaCoeffConst::update().
00018 
00019     - Second, there two additional random variables computed, the same as also
00020     computed by the number-fraction beta equation, see also
00021     DiffEq/NumberFractionBeta.h.
00022 
00023     In a nutshell, the equation integrated governs a set of scalars,
00024     \f$0\!\le\!X_\alpha\f$, \f$\alpha\!=\!1,\dots,N\f$, as
00025     \f[
00026        \mathrm{d}X_\alpha(t) = \frac{b_\alpha}{2}\left(S_\alpha - X_\alpha\right)
00027        \mathrm{d}t + \sqrt{\kappa_\alpha X_\alpha(1-X_\alpha)}
00028        \mathrm{d}W_\alpha(t), \qquad \alpha=1,\dots,N
00029     \f]
00030     with parameter vectors \f$b_\alpha = \Theta b'_\alpha > 0\f$, \f$
00031     \newcommand{\irv}[1]{\langle{#1^2}\rangle} \kappa_\alpha = \kappa' \irv{x} >
00032     0\f$, and \f$0 < S_\alpha < 1\f$. This is similar to DiffEq/Beta.h, but the
00033     parameters, \f$b\f$ and \f$\kappa\f$ constrained. Here \f$
00034     \newcommand{\irv}[1]{\langle{#1^2}\rangle}
00035     \newcommand{\irmean}[1]{{\langle{#1}\rangle}} \Theta = 1 - \irv{x} /
00036     [ \irmean{X} (1-\irmean{X}) ]\f$. The fluctuation about the mean, \f$
00037     \newcommand{\irmean}[1]{{\langle{#1}\rangle}} \irmean{X} \f$, is defined as
00038     usual: \f$ \newcommand{\irmean}[1]{{\langle{#1}\rangle}} x = X - \irmean{X}
00039     \f$, and \f$b'\f$ and \f$ \kappa'\f$ are user-specified constants. Also,
00040     \f$\mathrm{d}W_\alpha(t)\f$ is an isotropic vector-valued
00041     [Wiener process](http://en.wikipedia.org/wiki/Wiener_process) with
00042     independent increments. The invariant distribution is the joint beta
00043     distribution. This system of SDEs consists of N independent equations. For
00044     more on the beta SDE, see http://dx.doi.org/10.1080/14685248.2010.510843.
00045 
00046     Similar to the number-fraction beta SDE (DiffEq/NumberFractionBeta.h), in
00047     addition to integrating the above SDE, there are two additional functions
00048     of \f$ X_\alpha \f$ are computed as
00049     \f[ \begin{align}
00050       \rho(X_\alpha) & = \rho_{2\alpha} ( 1 - r'_\alpha X_\alpha ) \\
00051       V(X_\alpha) & = \frac{1}{ \rho_{2\alpha} ( 1 - r'_\alpha X_\alpha ) }
00052     \end{align} \f]
00053     These equations compute the instantaneous mixture density, \f$ \rho \f$, and
00054     instantaneous specific volume, \f$ V_\alpha \f$, for equation \f$ \alpha \f$
00055     in the system. These quantities are used in binary mixing of
00056     variable-density turbulence between two fluids with constant densities, \f$
00057     \rho_1, \f$ and \f$ \rho_2 \f$. The additional parameters, \f$ \rho_2 \f$
00058     and \f$ r' \f$ are user input parameters and kept constant during
00059     integration. Since we compute the above variables, \f$\rho,\f$ and \f$V\f$,
00060     and call them mixture density and specific volume, respectively, \f$X\f$,
00061     governed by the beta SDE is a number (or mole) fraction.
00062 
00063     _All of this is unpublished, but will be linked in here once published_.
00064 */
