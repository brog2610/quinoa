Detailed Description

template<class Stack, class eq, class param>
struct tk::grm::check_spikes< Stack, eq, param >

Check if the spikes parameter vector specifications are correct.

Spikes are used to specify sample-space locations and relative probability heights for a joint-delta PDF.
