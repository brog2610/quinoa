Detailed Description

Driver. The class below uses runtime polymorphism without client-side inheritance: inheritance is confined to the internals of the class below, inivisble to client-code. The class exclusively deals with ownership enabling client-side value semantics. Credit goes to Sean Parent at Adobe: https://github.com/sean-parent/sean-parent.github.com/wiki/ Papers-and-Presentations
