      66                 :            :  * \section Teuchos_ArrayRCP_Details Details

      68                 :            :  * Managing an array of objects is very different from managing a
      69                 :            :  * pointer to an individual, possibly polymorphic, object.  For
      70                 :            :  * example, while implicit conversions from derived to base types can
      71                 :            :  * be useful when dealing with pointers to single objects, they often
      72                 :            :  * cause problems when working with arrays of objects.  Therefore,
      73                 :            :  * this class contains those capabilities of raw pointers that are
      74                 :            :  * good when working with arrays of objects, but excludes those that
      75                 :            :  * are bad, such as implicit conversions from derived to base types.
      76                 :            :  * If you really do want a shared array with implicit conversions from
      77                 :            :  * derived to base types, you may use an <tt>ArrayRCP<RCP<T> ></tt>.

      82                 :            :  * class by setting the <tt>Teuchos_ENABLE_DEBUG:BOOL=ON</tt> CMake
      83                 :            :  * option when configuring your Trilinos build.  This option is off by
      84                 :            :  * default.  It incurs a significant performance penalty and so is not
      85                 :            :  * recommended for production builds.  Bounds checking requires that
      86                 :            :  * you always create ArrayRCP instances with the correct range.  For

      88                 :            :  * pointer, you are responsible for supplying the correct number of
      89                 :            :  * elements in the array.  Our bounds checking implementation does not
      90                 :            :  * attempt to replace memory debugging tools such as the Memcheck tool

      96                 :            :  * elements in the array:
