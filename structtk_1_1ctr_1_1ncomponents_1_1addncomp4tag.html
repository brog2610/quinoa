Detailed Description

template<typename... Tags>
template<typename tag>
struct tk::ctr::ncomponents< Tags >::addncomp4tag< tag >

Function object for computing the offset for a given tag (i.e., the sum of the number of components up to a given tag). This is used to index into the particle array and get to the beginning of data for a given differential equation system.
