Detailed Description

For testing boost::value_factory (runtime polymorphism using value semantics). The struct below uses runtime polymorphism without client-side inheritance: inheritance is confined to the internals of the struct below, inivisble to client-code. The struct exclusively deals with ownership enabling client-side value semantics. Credit goes to Sean Parent at Adobe: https://github.com/sean-parent/sean-parent.github.com/wiki/ Papers-and-Presentations
